# Windows CUDA 开发入门

## 1 Windows CUDA 版本切换

Windows10 系统中能同时安装多个 CUDA 版本。



切换 CUDA 版本的方法：

* 更改系统环境变量
  * Win + R  

  * 输入 sysdm.cpl

  * 系统属性界面  -> 高级 -> 环境变量 

  * 环境变量中的 Path -> 编辑 -> 将CUDA 版本上移
